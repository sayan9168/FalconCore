# Contributing to FalconCore

Thanks for your interest!

## How to Contribute
- Fork the repo
- Create branch: `git checkout -b feature/your-feature`
- Commit: `git commit -m 'Add your feature'`
- Push: `git push origin feature/your-feature`
- Open Pull Request

Welcome:
- Bug fixes
- New examples
- Documentation
- Feature ideas

Happy coding!
